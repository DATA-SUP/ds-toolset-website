# TOTP Toolset - website
Fully client-side version based on Token2 TOTP Toolset (Token2 TOTP Toolset - local), which can be run locally without accessing any libraries/resources on the Internet (including the QR image generation).

### Features
* convert hex seed to base32 format
* generate QR image based on hex or base32 seed key values
* generate random seed values (i.e. for programmable tokens or mobile app)
* verify the time drift with customizable skew value



## Usage
Visit https://qrcode.data-sup.fr/
